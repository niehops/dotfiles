# Hei's Dotfiles 🚀

Personal configuration for macOS terminal environment.

### Tools included:
- **Shells:** Fish, Nushell, Zsh
- **Editor:** Neovim
- **Terminal:** Ghostty
- **Multiplexer:** Tmux
- **Prompt:** Starship

### Quick Setup:
```bash
git clone <your-repo-url> ~/dotfiles
cd ~/dotfiles
./install.sh
