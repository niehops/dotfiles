# Hei's Dotfiles 🚀

Personal configuration for macOS terminal environment.

### Tools included:
- **Shells:** Fish, Nushell, Zsh
- **Editor:** Neovim
- **Terminal:** Ghostty
- **Multiplexer:** Tmux
- **Prompt:** Starship

### Quick Setup:
```bash
git clone https://github.com/niehops/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

### Tmux plugin Setup:
run Tmux
```bash
tmux
```

#### To install Tmux plugins
(Ctrl + a ) + I (capital)
