# Dotfiles 🚀

Personal configuration for macOS/Linux terminal environment.

### Supported OS:
- macOS
- Ubuntu/Debian/Pop
- Fedora
- Arch/Manjaro

### Tools included:
- **Shells:** Zsh
- **Editor:** Neovim
- **Terminal:** Ghostty
- **Multiplexer:** Tmux
- **Prompt:** Starship

### Quick Setup:
```bash
curl -fsSL https://raw.githubusercontent.com/niehops/dotfiles/main/install.sh | bash

zsh
```

### Tmux plugin Setup:
run Tmux
```bash
tmux
```

#### To install Tmux plugins
(Ctrl + a ) + i (capital)
